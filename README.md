# GB_XO-game_KS
Игра в крестики-нолики в рамках курса «Архитектуры и шаблоны проектирования»

 Игра поддерживает три режима:
 - Два игрока
 - Игра против компьютера
 - Игра двух игроков в пять ходов одновременно по очереди
